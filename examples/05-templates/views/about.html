How Templates Work in Expressify

Templates in Expressify are rendered using the following pattern:

# Set up the template engine
app.set('view engine', 'jinja2')
app.set('views', './views')

# Render a template with data
@app.get('/route')
def handler(req, res):
    res.render('template.html', {
        'title': 'Page Title',
        'data': some_data
    })
